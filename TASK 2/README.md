# 🎮 Tic-Tac-Toe AI using Minimax Algorithm

A console-based Tic-Tac-Toe game developed in Python where a human player competes against an AI opponent powered by the **Minimax Algorithm**. The AI evaluates every possible game state to make the optimal move, making it impossible to defeat.

---

## 📌 Project Overview

This project demonstrates the implementation of Artificial Intelligence concepts through the classic Tic-Tac-Toe game. The AI uses the Minimax algorithm, a decision-making algorithm widely used in game theory, to choose the best possible move in every situation.

The project was developed to strengthen understanding of:

- Game Theory
- Artificial Intelligence
- Search Algorithms
- Recursion
- Backtracking
- Decision Trees
- Python Programming

## 🧠 Algorithm Used

### Minimax Algorithm

The AI explores every possible future game state before making a move.

Scoring System:

| Game State | Score |
|------------|------:|
| AI Wins | +1 |
| Draw | 0 |
| Human Wins | -1 |

The algorithm recursively evaluates every possible move and selects the one with the highest score.
